PROJECT TITLE:
SmartSeat — Real-Time Library & Classroom Seat Occupancy Detection
using YOLOv8 on Raspberry Pi 5

SHORT DESCRIPTION:
SmartSeat is an edge AI system that detects occupied and vacant seats
in library and classroom environments using a pretrained YOLOv8n model
(yolov8n.pt, COCO weights) deployed on a Raspberry Pi 5 (16GB) via
TensorFlow Lite. The system achieves 14–15 FPS with INT8 quantization
at only 34% CPU usage — compared to 5–6 FPS and 61% CPU in the
baseline mode — and displays real-time occupancy on a local web
dashboard. Validated across three real-world environments: open lounge,
classroom rows, and computer lab workstations.

MEASURED PERFORMANCE SUMMARY (Raspberry Pi 5)
================================================================

Mode          | FPS    | CPU Before | CPU During | Model Size
--------------+--------+-----------+-----------+-----------
Normal        | 5–6    | 4%         | 61%        | —
Float32 TFLite| 7–8    | 3%         | 36%        | ~12–14 MB
FP16 TFLite   | 5–6    | 3%         | 35%        | ~6–7 MB
INT8 TFLite   | 14–15  | 3%         | 34%        | ~3–4 MB  ← deployed

Key finding: INT8 quantization delivers ~3× faster inference and
~44% lower CPU usage compared to the baseline normal mode.
FP16 provides no speed benefit on RPi 5 (no FP16 hardware acceleration).
